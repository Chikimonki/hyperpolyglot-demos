# PolyRaft - Distributed Polyglot Consensus

Distributed consensus system demonstrating:
- Leader election (Raft-inspired)
- Distributed computation (parallel Fibonacci)
- Fault tolerance (node failure/recovery)
- State replication (consistent cluster state)

## Stack

- Zig: High-performance compute nodes
- Go: Cluster coordination
- Julia: Analytics aggregation
- LuaJIT: Consensus protocol
- Python: Monitoring dashboard
